### Plugin description

* Check if roi
* Find nucleus with Stardist
* Find HA-ORF1P with cellpose
* Find nuclei colocalized with HA-ORF1P cells
* Compute nucleus form factors and intensity for nucleus / HA-ORF1P+- cells
